# SEO Off-Page Automation Runbook

Этот документ описывает, как быстро и безопасно наращивать внешние упоминания без "серого" линкбилдинга.

## 1) Подготовка
- Заполнить `seo/profile-data.json`.
- Актуализировать список площадок в `seo/backlink-prospects.csv`.
- Выбрать первую волну: все `tier=1`, затем `tier=2`.

## 2) Размещения (полуавтомат)
- Использовать шаблоны из `seo/submission-templates.md`.
- Для каждой площадки фиксировать:
  - `status` (`todo` -> `in_progress` -> `placed`),
  - `profile_url`,
  - `placed_url`,
  - `requires_manual_step`.
- Когда нужна верификация (почта/SMS/капча), выполнять вручную.

## 3) Контроль качества
- Проверять, что ссылки ведут на HTTPS-версию сайта.
- Не ставить одинаковый коммерческий анкор массово.
- Сохранять баланс: брендовые анкоры + естественные вхождения.

## 4) Контроль индексации и эффекта
- После размещения обновить карту сайта и отправить приоритетные URL на переобход в панелях вебмастеров.
- Каждые 7-14 дней:
  - сверять индексируемость страниц-доноров,
  - проверять динамику внешних ссылок в Яндекс.Вебмастере,
  - отслеживать рост показов/кликов по бренд-запросам.

## 5) Команды для трекинга
- `python3 seo/manage_backlinks.py summary`
- `python3 seo/manage_backlinks.py start --platform "vc.ru"`
- `python3 seo/manage_backlinks.py mark-placed --platform "vc.ru" --placed-url "https://vc.ru/..."`

## 6) Стоп-факторы (не делаем)
- Массовые прогоны по тысячам каталогов.
- Покупка пакетов ссылок с нетематических площадок.
- Переспам анкорами в стиле "купить консалтинг дешево".
