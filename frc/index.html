Шаблоны мета-тегов

Шаблоны ниже нужны разработке и контенту как стартовая рамка. Конкретные фразы уточняются после семантики и финального ассортимента. 

<!-- Главная -->
<title>Универмаг «Россия» — официальный интернет-магазин мерча</title>
<meta name="description" content="Одежда и мерч Национального центра «Россия»: коллекции, футболки, свитшоты, кепки и подарочные товары с доставкой по России.">

<!-- Категория -->
<title>{Категория} Универмаг «Россия» — купить с доставкой</title>
<meta name="description" content="{Категория} из коллекций Универмага «Россия»: актуальные модели, размеры, цены и доставка по России.">

<!-- Карточка товара -->
<title>{Название товара} — купить в Универмаге «Россия»</title>
<meta name="description" content="{Название товара}: описание, фото, размеры, цена и условия доставки по России.">

Canonical и фильтры

Индексировать только фильтры, которые дают самостоятельный спрос и имеют уникальный текст/товарную выдачу. Остальные комбинации закрывать через canonical или noindex. 

<!-- Индексируемая категория -->
<link rel="canonical" href="https://example.ru/catalog/futbolki/">

<!-- Неиндексируемая комбинация фильтров -->
<meta name="robots" content="noindex, follow">
<link rel="canonical" href="https://example.ru/catalog/futbolki/">

Schema.org

Минимальный набор: Organization, WebSite/SearchAction, BreadcrumbList, Product, Offer, ImageObject для товарных изображений. 

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Product",
  "name": "{Название товара}",
  "image": ["https://example.ru/images/{product}.jpg"],
  "brand": {
    "@type": "Brand",
    "name": "Универмаг «Россия»"
  },
  "offers": {
    "@type": "Offer",
    "priceCurrency": "RUB",
    "price": "{Цена}",
    "availability": "https://schema.org/InStock",
    "url": "https://example.ru/product/{slug}/"
  }
}
</script>

Robots.txt и sitemap

Служебные сценарии закрываются от индексации. В sitemap попадают только канонические страницы, которые должны ранжироваться. 

User-agent: *
Disallow: /cart/
Disallow: /checkout/
Disallow: /account/
Disallow: /favorites/
Disallow: /search/
Allow: /

Sitemap: https://example.ru/sitemap.xml

Требования к карточке товара

  • Уникальный H1 с названием товара.
  • Цена, наличие, размеры, цвета и состав в текстовом HTML.
  • Фото с alt по шаблону: «{Название товара} — Универмаг Россия».
  • Блок доставки и возврата без скрытия важного текста в изображениях.
  • Хлебные крошки: Главная / Каталог / Категория / Товар.
  • Product schema с Offer и актуальной доступностью.
